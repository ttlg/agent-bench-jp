# タスク: ファイル行数カウントCLIツール

TypeScriptで、指定ディレクトリ内のファイルの行数をカウントするCLIツールを作成してください。

## 要件

- コマンド名: `lc`（line count）
- 引数: ディレクトリパス（省略時はカレントディレクトリ）
- オプション:
  - `--ext <拡張子>`: 特定の拡張子のファイルのみカウント（例: `--ext .ts`）
  - `--sort`: 行数の降順でソート
  - `--total`: 合計行数も表示
- 出力例:
  ```
  src/index.ts    150
  src/utils.ts     82
  src/cli.ts       45
  ─────────────────
  合計             277
  ```
- 隠しファイル（.で始まるファイル）と node_modules は除外すること
- npm script または npx で実行できるようにすること
