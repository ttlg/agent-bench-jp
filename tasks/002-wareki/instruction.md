# タスク: 和暦変換CLIツール

TypeScriptで、西暦と和暦を相互変換するCLIツールを作成してください。

## 要件

- コマンド名: `wareki`
- サブコマンド:
  - `wareki to-wareki <日付>`: 西暦 → 和暦に変換
  - `wareki to-seireki <和暦文字列>`: 和暦 → 西暦に変換
- 対応元号: 明治・大正・昭和・平成・令和
- 入力形式:
  - 西暦: `2026-03-10` (ISO 8601)
  - 和暦: `令和8年3月10日`
- 出力例:
  ```
  $ wareki to-wareki 2026-03-10
  令和8年3月10日（火）

  $ wareki to-seireki 令和8年3月10日
  2026-03-10 (Tuesday)
  ```
- オプション:
  - `--format short`: 短縮表記（例: `R8.3.10`）
- 元号の境界日を正しく処理すること（例: 1989-01-07 は昭和64年、1989-01-08 は平成元年）
- 不正な日付にはエラーメッセージを表示すること
- 外部の日付ライブラリを使わず、自前で実装すること
- npm script または npx で実行できるようにすること
